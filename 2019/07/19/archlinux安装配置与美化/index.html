Archlinux安装配置与美化

给出详细的archlinux安装图文教程，初次接触archlinux的朋友能够按照教程完成arch安装。在后文给出一些archlinux的配置和美化攻略。本文长期更新。

另外，安利大家使用archlinux。笔者先后使用了ubuntu、debian、openSUSE、deepin、elementary OS、Nitrix、Zorie OS、Cent OS等Linux发行版，最后投奔到archlinux麾下。

archlinux优点和缺陷，网络上有很多的资料，大家可以去参考。但是，告诉朋友的是，任何一个发行版，都有自己的优点，也有自己的缺点。没有必要互相攻讦或者互相吹捧。这么多linux发行版的尝试，对笔者而言，只能说虚耗时光，倒是最后停留在arch之后的时间里，才对linux作了认真的学习，有了一些收获。所以，推荐arch。

正文开始！

u盘启动工作制作

下载archlinux镜像文件。

到archlinux官网下载页面，找到下图标注的地方，点击下载(网页一直下拉即可找到)。

下载源

清华源

下载u盘制作工具

U盘制作工具推荐使用Ultraiso.

刻入u盘

  1. 打开UltraISO。
  2. 点击文件菜单，打开刚下载的archlinux的iso文件。
  3. 点击启动菜单，选择写入硬盘影响.
  4. 硬盘驱动器选择自己准备好的u盘（U盘的文件会被清理，请提前做好备份）
  5. 写入方式选择RAW。（一定要选择此选项，其它选项好像不能成功）

开始安装

开机

关闭电脑，插入u盘，开机之后，进入BISO面板（一般电脑在开机的时候按住F12键，不同电脑快捷键不同，如果F12不能成功，请网上查找对应型号的快捷键。）

选择u盘启动

此时，在root用户权限下。

wifi链接

接下来连接网络，输入wifi-menu命令，按回车。在弹出框中选择自己的wifi并输入密码。

wifi-menu

ping -c 4 www.baidu.com

按回车，如果没有报错，而是输出了ping回来的信息，说明链接成功。

执行

镜像站文件编辑

linux终端一般使用vim作为编辑器。先安装vim.

输入pacman -S vim，按回车，会成功安装vim编辑器。

接下来编辑镜像源文件。

输入vim /etc/pacman.d/mirrorlist，然后按回车，打开配置文件.

按/China命令，回车找到第一个China源。按dd剪切源文件，按gg返回到首行，按下移光标，将光标切换到’# Germany’行首，按p粘贴。重复上面的动作，将China源移到前面。在镜像文件中，有多个China的链接，一般选择清华源或者中科大源即可。

确保源文件修改正确，按:wq，按回车键退出文件编辑。

分区

分区有很多的方案。这里推荐一种新手易于使用的方案。

首先查看当前磁盘信息：

执行lsblk命令，按回车，输出当前电脑磁盘信息。

使用fdisk命令分区。

执行：fdisk /dev/sda,其中sda是前面输出的磁盘名称，按回车。

输入g（目的是建立分区表，读者不用关心它是什么）

输入n（添加一个新的分区，请阅读提示信息）

输入+2G，按回车。

这样建立了一个2G大小的磁盘空间。

磁盘格式化

现在要给不同的分区加载不同的文件系统。

执行mkfs.fat -F32 /dev/sda1，给第一个分区加载fat文件系统。

执行mkfs.ext4 /dev/sda2， 给第二个分区加载ext4文件系统。

执行mkswap /dev/sda3, 将第三个分区作为交换空间

执行mkfs.ext4 /dev/sda4，给第四个分区加载ext4文件系统。

文件挂载

分区准备好之后，我们需要将镜像文件挂载到相应的分区。

执行mount /dev/sda2 /mnt。

执行mkdir /mnt/boot。

执行mkdir /mnt/home。

执行mkdir /mnt/boot/EFI。

执行mount /dev/sda1 /mnt/boot/EFI。

执行mount /dev/sda4 /mnt/home。

执行swapon /dev/sda3。

至此，文件挂载成功。

基本系统安装

执行pacman -i /mnt base base-devel。

开始安装。

配置fstab

执行genfstab -U /mnt >> /mnt/etc/fstab。

执行cat /mnt/etc/fstab查看， 如果显示如下，说明配置成功。

本地语言配置

首先执行arch-chroot /mnt /bin/bash，切换到新系统。

执行vim /etc/locale.gen。

利用/ en_US.UTF-8 UTF-8，去掉前面的注释符号（按x键可以删除）

利用/ zh_CN.UTF-8 UTF-8，去掉前面的注释符号。

执行locale-gen

执行echo LANG=en_US.UTF-8 > /etc/locale.conf。设置系统语言为英文。如果需要系统语言为中文，就执行echo LANG=zh_CN.UTF-8 > /etc/locale.conf。

时区设置

执行ln -sf /usr/share/zoneinfo/Asia/Shanghai /etc/localtime。

设置硬件时间：

执行hwclock --systohc --utc。

安装系统引导

执行pacman -S grub efibootmgr(默认为EFI引导)

执行grub-install --target=x86_64-efi --efi-directory=/boot/EFI --recheck。

执行grub-mkconfig -o /boot/grub/grub.cfg。

切换到新用户

arch-chroot /mnt /bin/bash

新建用户

首先设置root用户密码。

执行passwd。输入root用户的密码。

执行useradd -m -g users -s /bin/bash name。其中name改成自己的用户名字。

执行passwd name，设置用户密码。

执行sudo vim /etc/suders。光标下移找到root ALL=(ALL) ALL，在下面添加name ALL=(ALL) ALL一行。（按a进入插入模式）

网络配置

执行sudo pacman -S dhcpcd networkmanager，安装网络驱动。

执行sudo systemctl enable dhcpcd。

执行sudo pacman -S iw wpa_supplicant dialog，安装无线驱动和管理工具。

如果是宽带用户，需要配置下面两个语句：

执行sudo pacman -S rp-pppoe# pppoe-setup

执行sudo systemctl start adsl

安装显卡驱动

执行lspci | grep VGA，查看电脑的显卡。按照显卡安装相应的驱动：

执行sudo pacman -S 驱动名称

常见的驱动有：

显卡 驱动名称
通用 xf86-video-vesa
intel xf86-video-intel
Geforce7 xf86-video-nouveau
Geforce6/7 xf86-video-304xx

安装x窗口

执行pacman -S xorg，选择所有默认软件，一路回车，完成xorg窗口安装。

执行pacman -S xf86-input-synaptics，安装触摸板驱动（台式机不需要）。

执行pacman -S ttf-dejavu wqy-microhei wqy-zenhei，完成中文字体支持（不然会出现中文字体变成方框的现象）。

安装桌面环境

安装plasma

执行pacman -S plasma，安装plasma桌面环境。

执行pacman -S konsole，安装kde的控制台。

执行pacman -S dolphin， 安装kde文件资源管理器。

执行pacman -S kde-applicantions，安装kde常用工具套装。

执行systemctl enable sddm，启动sddm显示管理器。

执行sysytemctl enable NetworkManager，启动网络管理。

执行pacman -S plasma-nm，安装kde网络管理前端工具（图形界面）。

执行reboot，重新启动。

至此，安装基本完成。

安装gnome

执行pacman -S gnome，安装gnome桌面。

执行pacman -S gnome-tweaks，安装gnome图形配置工具。

执行pacman -S alacarte，安装gnome桌面菜单编辑器。

执行sysytemctl enable gdm，启动gnome窗口管理服务。

执行sysytemctl enable NetworkManager，启动网络管理服务。

执行reboot，重新启动。

源更换

更换archlinuxcn源，安装一些中国用户常用的软件。

执行sudo vim /etc/pacman.conf

在文件最后添加：

[archlinuxcn]
SigLevel=Never
Server =https://mirrors.ustc.edu.cn/archlinuxcn/$arch

按Esc键，接着输入:wq保存。

执行sudo pacman -Syu，即可更新源。

输入法安装

推荐使用fcitx输入框架。

执行

sudo pacman -S fcitx-im fcitx-configtool fcitx-sogoupinyin fcitx-googlepinyin

然后打开文件（没有就创建一个）：

vim ~/.pam_environment

然后在里面输入一下内容：

GTK_IM_MODULE=fcitx
QT_IM_MODULE=fcitx
XMODIFIERS=@im=fcitx

然后打开fcitx-configtool进行配置输入法和快捷键即可。
