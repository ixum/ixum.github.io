Pandas常用API汇总

汇总pandas常用的API，介绍其功能和使用方法

pandas提供的对象

  1. Series
  2. DataFrame
  3. data_range
  4. Timestamp

DataFrame常用API

生成DataFrame

df = DataFrame(np.random.randn(6, 4), index=data_range('20190704', period=6), columns=list('ABCD') )

df2 = df.copy()  # 创建一个新对象

查看数据

df.head()  # 查看前5行数据
df.head(3) # 查看前3行数据
df.tail()  # 查看后5行数据
df.tail(3) # 查看后3行数据

df.descibe() # 对数据做简单分析，给出平均数，分位数，统计数，最大最小值等
df.info()    # 返回数据主要信息：包括行列数量，缺失值数量，列数据类型等

排序

df.sort_index(axis=1,ascending=False) # 按照坐标轴排序，0-行；1-列；
df.sort_values(by='B')   # 按照某一行的值排序
df.T                     # 行列转置

索引

pandas二维表可以按照标签，位置来索引。

按照标签索引可以使用at和loc方法，也可以使用[]来切片。

按照位置索引可以使用iat和iloc方法。

其中loc是从低维到高维一级一级找，at直接定位到具体的scalar。因此，如果是行、列或者数据块的提取，可以使用loc方法。直接指定到具体值，就使用at方法。

# 指定具体值
df.A
df['A']  # 索引A列, 返回一个Series对象
df[2:4]  # 切出2、3行
df.at[index, columns] # 按照label指定具体值
df.iat[0, 1]          # 按照位置指定具体值

# 索引数据块
df.loc[:,'D']         # 按照label索引一列
df.loc[index, columns] # 按照label指定具体值
df.loc[:,['A','B']]    # 按照label索引指定列
df.loc[2:4,['A','B']] 
df.iloc[:, 1:5]       # 按照位置索引某几列
df.iloc[3]            # 指定第三行
df.iloc[1, 2]         # 按照位置指定具体值
df.iloc[[1, 2, 4], [0, 2]] 

# 按照bool值索引
df[df>0]              # 按照bool值选择
df[df.A>0]
df[df['B'].isin([2019, 2018, 2016])]  # 使用isin（）判断

赋值

赋值与索引结合密切。

s1 = pd.Series(list(),index=...)
df['F'] = s1  # 添加新列

常见属性

df.index             # 返回index对象
df.columns           # 返回columns对象
df.reindex()         # 重新赋index、columns
df.dropna(how='any') # 剔除所有有nan的行
df.fillna(value=5)   # 将nan项赋值为5
pd.isna(df)          # 返回判断结果——二维bool表
df.mean(0)           # 返回axis=0的轴上的均值

运算符

pandas运算符具有广播性质，是针对元素进行的。

df = pd.DataFrame(np.random.randn(6, 4), index= pd.date_range('20190714',periods=6), columns=list('ABCD'))
s = pd.Series([1, 2, 3, 4, np.nan, 6, 8], index= pd.date_range('20190714',periods=6)).shift(2)

df.sub(s, axis='index')   # 按照index对齐并进行减法
df.add()
df.mul()
df.div()
df.truediv()
df.floordiv()
df.mod()
df.pow()

df.apply(np.cumsum)       # 按照行累加。可以使用axis=1来控制按照列累加。对于每一个元素进行计算，np.cumsum列表累加函数，

df.apply(lambda x: x.max()-x.min(), axis = 1)

连接

df.concat([], axis=0)
df.merge()
df.append(s, ignore_index=True)

分组

df.groupby('A')     # 按照A分组
df.goupby(['A', 'B'])  # 按照A+B的组合分组

Series常用API

生成Series对象

s = pd.Series(np.random.randn(0, 7, size=10))

常用属性

s.value_counts()   # 统计数字出现的次数
s.str              # 将元素转换成字符串
